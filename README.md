# primera-api

Ejercicio de primeros pasos consumiendo una API REST desde el navegador con JavaScript vanilla, usando la API pública de Rick and Morty para pintar personajes en HTML.

## Contexto

Ejercicio del curso de consumo de APIs de Platzi. Se practica `fetch`, promesas y renderizado dinámico de tarjetas en el DOM a partir de los datos de la [API de Rick and Morty](https://rickandmortyapi.com/api).

## Tecnologías

- HTML5
- CSS3
- JavaScript (vanilla, Fetch API)
- [Rick and Morty API](https://rickandmortyapi.com/)
